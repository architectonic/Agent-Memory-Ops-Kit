# Ecosystem

This folder tracks technologies, tools, and documentation references relevant to agent memory operations.

It is a curated index, not an encyclopedia.

## Files

- `tool_categories.md`
- `docs_index.md`
- `stack_patterns.md`

## Rules

- Prefer stable official docs.
- Avoid link dumps.
- Record why a tool matters to agent memory workflows.
- Mark unstable or fast-changing information as review-needed.
- Do not include private tool credentials, private workspace links, or internal systems.
