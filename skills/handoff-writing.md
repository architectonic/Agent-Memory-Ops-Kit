# Handoff Writing Skill

## Trigger

Use when ending a session, pausing work, transferring work to another agent, or leaving a task incomplete.

## Inputs

- Task request.
- Current state.
- Changed files.
- Verification results.
- Open questions.

## Procedure

1. State the task in one sentence.
2. List completed work.
3. List changed files.
4. Record verification performed.
5. Identify blockers and open questions.
6. State the next concrete step.
7. Avoid narrative filler.

## Verification

A new agent should be able to continue without rereading the full conversation.

## Failure Modes

- Writing a vague summary.
- Omitting changed files.
- Hiding uncertainty.
- Failing to record verification or blockers.
