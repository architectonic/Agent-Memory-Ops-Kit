# Minimal Example

A minimal agent-ready repository.

```text
AGENTS.md
GOALS.md
HANDOFF.md
DECISIONS.md
MEMORY.md
```

## Purpose

Demonstrate the smallest useful operating memory system.

## Workflow

1. Read AGENTS.md.
2. Review goals.
3. Perform work.
4. Record decisions.
5. Update handoff.
6. Update memory only when durable knowledge was created.

## Rule

Do not create memory entries for temporary conversation context.
