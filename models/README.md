# Models

Models define reusable file patterns for agent memory systems.

Use these to create project-specific files without inventing structure each time.

## Available Models

- `skill_model.md`
- `goal_model.md`
- `dream_cycle.md`
- `wiki_model.md`
- `ontology_model.md`
- `preferences_model.md`
- `decision_model.md`

## Rule

Models are templates. They are not project truth until filled with project-specific evidence or human confirmation.
